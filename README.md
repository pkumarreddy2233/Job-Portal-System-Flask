# Job Portal System Using Flask and SQLite

## Overview
A full-stack Job Portal System developed using Python, Flask, HTML, CSS, and SQLite.

## Features
- User Registration
- User Login
- Dashboard
- Post New Jobs
- View Available Jobs
- SQLite Database Integration
- Responsive UI

- CSS
- SQLite
